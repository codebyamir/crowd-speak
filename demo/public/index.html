Setup required for this demo to work

Build the JS client
  1. cd client
  2. npm install
  3. npm run build
There should now be a file at client/dist/crowdspeak.min.js
Start the API server
  1. cd api
  2. npm install
  3. npm run start
This should start the API at http://localhost:3000 with the minified JS file at http://localhost:3000/crowdspeak.min.js
